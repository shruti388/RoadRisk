<h1>Road Risk Prediction System</h1>
<hr>

<h2>📖 Project Overview</h2>
    Road Risk Prediction System is a web-based application that helps users to analyze road safety based on real-time weather conditions. The system predicts wheather it is safe to travel by evaluating factors        like temperature, humidity, and wind Speed.
    <hr>

<h2>Objective</h2>
    The main objective of this project is to:
    <br>
      ⬤ Predict road risk using weather data
      <br>
      ⬤ Provide safer travel suggestions
      <br>
      ⬤ Reduce accidents caused by bad weather conditions
      <br>
      ⬤ To calculate and display risk levels (Low, Medium, High)
      <hr>
